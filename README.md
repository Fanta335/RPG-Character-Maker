# Usage
入力する情報は以下のとおりです。
- 名前
- 性別
- 職業
- 性格

ステータスは以下のパラメータで出力されます。
- 力
- 素早さ
- 体力
- 賢さ
- 運の良さ

キャラクターの画像は、性別および職業に応じて変化します。  
ステータスの値は、職業および性格に応じて変化します。
